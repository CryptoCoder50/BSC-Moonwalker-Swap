# 🚀 MoonWalker Swap Exchange

[MoonWalkerSwap](https://bsc-swap.moonwalker.network) is an automated market maker (“**AMM**”) that allows two tokens to be exchanged on the [Binance Smart Chain](https://www.binance.org/en/smartChain) (BSC). It is fast, cheap, and allows anyone to participate.

This repo is responsible for the **exchange** interface of the AMM: [Swap-MoonWalker.Network](https://bsc-swap.moonwalker.network)
